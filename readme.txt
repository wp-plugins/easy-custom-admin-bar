Easy Custom Admin Bar is the most easy and simple plugin for customizing your WordPress Dashboard Admin Bar.

== Description ==

When developing a WordPress website, maybe for client or for personal project, you may need to hide the WordPress logo, Add Custom Menus, Change color and lots of other changes in admin bar. By using this plugin, you can do it in just couple of clicks. In this free plugin, currently you can remove the WordPress logo and add custom menus which supports multi-level, that means you can add submenu in your custom menu.

You can also change color and lots of other stuffs in the paid version.

**Features**<br>
1. Add Unlimited Custom Menu<br>
2. Add Unlimited Submenu<br>
3. Remove WordPress Logo<br>
4. No coding required to customize<br>
5. Very easy user interface<br>
6. Beautiful Design<br>
7. Simple to use


== Installation ==

**Way 1:**<br>
1. Download the plugin file<br>
2. Unzip the file into a folder on your hard drive<br>
3. Upload via FTP into your plugins (/wp-content/plugins/) folder<br>
4. Activate the plugin from your dashboard<br>

**Way 2:**<br>
1. From Dashboard > Plugins > Add New > Upload the plugin<br>
2. Activate the plugin<br>
